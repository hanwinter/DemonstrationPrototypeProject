====================================================
一、页面定位
====================================================

该页面属于医院信息系统中的专业工作台页面，用于：

1. 展示OCR识别的检验报告图片；
2. 展示OCR识别结果；
3. 人工审核OCR结果；
4. 将OCR结果与系统标准检验项目进行匹配；
5. 最终导入孕产妇保健管理系统。

设计目标：

- 高效录入
- 高密度信息展示
- 适合医院长期使用
- 支持1920×1080显示器最佳体验
- 支持更大分辨率显示器
- 支持表格个性化调整

====================================================
二、页面整体布局
====================================================

页面采用：

Workbench（工作台模式）

整体布局：

┌──────────────────────────────────────────────┐
│ 顶部患者信息区                                │
├──────────────────────────────────────────────┤
│ OCR图片区 │ OCR校验工作台                     │
├──────────────────────────────────────────────┤
│ 底部操作按钮                                  │
└──────────────────────────────────────────────┘

====================================================
三、页面宽度规范
====================================================

要求：

整体内容区域居中显示

不要左对齐

配置：

max-width: 1800px;
min-width: 1600px;
margin: 0 auto;

行为：

1920显示器：
完整显示

2560显示器：
两侧留白
内容居中

小于1600：
出现横向滚动条

====================================================
四、左右工作区布局
====================================================

默认比例：

左侧OCR原图区域：
30%

右侧校验工作台：
70%

实现要求：

使用可拖拽Split Pane

用户可拖动中间分隔线

支持：

30% : 70%
20% : 80%
10% : 90%
0% : 100%

当左侧宽度拖到最小时：

自动折叠OCR区域

显示展开按钮

再次点击可恢复

====================================================

展示：

建档医院
