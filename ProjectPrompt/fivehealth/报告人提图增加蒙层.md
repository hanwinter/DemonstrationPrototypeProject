修改范围：
只修改报告页面中间卡片左侧“人体健康画像”区域，不修改页面顶部学生信息卡、综合评估卡、右侧说明卡片、下方指标卡片和底部导航。

当前问题：
1. 男孩/女孩人物 PNG 显示偏小。
2. 当前彩色圆点标记位置不准确，视觉效果不好。
3. 暂时不使用图片蒙层，先用 CSS 半透明区域覆盖实现。

图片路径：
- 男孩：ProjectImage\男孩.png
- 女孩：ProjectImage\女孩.png

实现目标：
1. 根据学生性别动态展示男孩或女孩 PNG。
2. 将人物图整体放大，主体高度尽量接近右侧说明卡片高度。
3. 删除当前5个悬浮彩色圆点 marker。
4. 使用 CSS 半透明 overlay 区域表示五健部位。
5. overlay 根据学生异常指标动态显示。
6. 当前右侧说明卡片展示的健康项对应 overlay 重点高亮。
7. 保持页面整体布局不变。

建议结构：
body-map-container 作为 position: relative 的定位容器；
人物图片使用 img 标签；
五健 overlay 使用 absolute 定位叠加在图片上方。

推荐代码结构：

<div class="body-map-container">
  <div class="body-map-bg"></div>
  <img class="body-image" :src="bodyImage" alt="人体健康画像" />

  <div
    v-for="item in visibleOverlays"
    :key="item.code"
    class="health-overlay"
    :class="[item.className, { active: item.code === activeHealthCode }]"
  ></div>
</div>

bodyImage 根据性别返回：
- 男：ProjectImage\男孩.png
- 女：ProjectImage\女孩.png
- 未知：默认 ProjectImage\男孩.png
