继续优化五健家长端报告页面中的“五健专项详情底部弹框”。

本次只修改点击“查看详情 >”后弹出的底部详情弹框，不修改报告页主页面结构，不修改五健健康画像卡主体布局，不修改人体图 overlay 高亮逻辑，不修改底部导航。

一、当前问题

当前底部弹框已经支持五健专项切换、异常详情展示和健康建议展示，但还存在以下问题：

1. 弹框顶部层级不够清晰。
   现在是“当前专项标题 + 风险等级”在最上面，“五健切换标签”在下面。
   这会让信息层级有些反。更合理的结构应当是：
   - 第一层：五健切换标签（一级导航）
   - 第二层：当前选中专项标题 + 风险等级 + 关闭按钮（当前内容标题）

2. “当前结论、异常指标、健康建议”三个模块的左侧起点和层级关系不够统一。
   目前视觉上会让“异常指标、健康建议”看起来像“当前结论”的二级内容，但它们实际上应该是并列的一级信息块。

3. 弹框高度会随着不同专项内容多少而变化，造成切换时弹框整体跳动，影响阅读体验。
   应将弹框设置为固定高度，内容超出时在弹框内部滚动。

二、修改目标

1. 将五健专项切换标签移动到弹框最顶层。
2. 将当前专项标题 + 风险等级 + 关闭按钮放到切换标签下方。
3. 统一“当前结论、异常指标、健康建议、检查信息”四个模块的左对齐方式和层级。
4. 底部弹框使用固定高度，不因专项内容不同而变化。
5. 弹框内部内容区域支持滚动。
6. 保留当前已有功能：
   - 点击切换五健专项；
   - 弹框内容随 activeHealthCode 更新；
   - 主页面右侧按钮选中态同步；
   - 人体图 overlay 同步；
   - 点击关闭按钮关闭弹框。

三、顶部结构调整

当前顶部大致为：

体重    中风险    ×
[视力高] [体重中] [脊柱低] [口腔正常] [心理待填]

请改成：

[视力高] [体重中] [脊柱低] [口腔正常] [心理待填]

体重    中风险                                  ×

也就是：
1. 五健专项切换标签放最上方；
2. 当前选中专项标题放在标签下方；
3. 标题行右侧保留关闭按钮；
4. 当前标题使用 activeHealth.navName，例如“视力、体重、脊柱、口腔、心理”；
5. 风险等级标签放在标题右侧；
6. 标题区样式保持简洁稳重，不要过高。

推荐结构：

<div class="sheet-health-tabs">
  <button
    v-for="item in healthItems"
    :key="item.code"
    class="sheet-health-tab"

    @click="activeHealthCode = item.code"
  >
    <span class="tab-name">{{ item.navName }}</span>
    <span class="tab-status">{{ getShortStatus(item.status) }}</span>
  </button>
</div>

<div class="sheet-header">
  <div class="sheet-title-row">
    <span class="sheet-title">{{ activeHealth.navName }}</span>
    <span class="risk-tag">{{ activeHealth.status }}</span>
  </div>
  <button class="sheet-close" @click="showHealthDetailSheet = false">×</button>
</div>

四、内容模块左对齐与层级统一

以下四个模块应当被设计成并列一级内容块，而不是上下看起来有主次嵌套：

1. 当前结论
2. 异常指标
3. 健康建议
4. 检查信息

要求：
1. 四个模块的左侧起点一致；
2. 模块标题样式一致；
3. 模块内容区域宽度一致；
4. 每个模块之间上下间距一致；
5. 不要让“异常指标、健康建议”看起来像“当前结论”的二级内容；
6. 健康建议虽然有浅色背景卡片，但也应与其他模块在左边界上对齐；
7. 异常指标不要再显示成插入式偏移条块，应与其他模块平级。

<div class="sheet-body">

    <div class="block-title">检查信息</div>

      </div>
    </div>
  </section>
</div>

五、固定弹框高度

当前弹框高度会随着内容变化。请改为固定高度。

要求：
1. 弹框整体高度固定，不因切换不同专项而改变高度；
2. 推荐高度：76vh 左右；
3. 可设置：
   - height: 76vh;
   - max-height: 80vh;
   - min-height: 72vh;
4. 弹框内容区域使用内部滚动；
5. 弹框头部（切换标签 + 标题区）固定在顶部；
6. 内容区域单独滚动；
7. 切换专项时弹框不要跳高或缩短；
8. 关闭弹框后页面背景仍恢复正常滚动。

推荐结构：

<div class="health-detail-sheet">
  <div class="sheet-top-fixed">
    <div class="sheet-health-tabs">...</div>
    <div class="sheet-header">...</div>
  </div>

  <div class="sheet-scroll-body">
    <!-- 当前结论、异常指标、健康建议、检查信息 -->
  </div>
</div>
