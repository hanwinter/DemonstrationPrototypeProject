继续优化五健专项详情底部弹框的内容对齐问题。

本次只修改弹框内部内容区的对齐和间距，不修改主页面结构、不修改五健切换标签、不修改弹框交互逻辑、不修改底部导航。

当前问题：
1. “当前结论、异常指标、健康建议”的标题左侧对齐已经基本正确。
2. 但“健康建议”的内容块没有与“当前结论正文、异常指标内容块、检查信息表格”左侧对齐。
3. 健康建议浅青色背景块横向过宽，贴近弹框边缘，看起来像色带。
4. 异常指标、健康建议等内容块与手机边框之间需要保留统一安全边距。

修改目标：
1. 弹框内容区建立统一内容栅格。
2. 当前结论、异常指标、健康建议、检查信息四个模块的标题和内容块都从同一左边界开始。
3. 异常指标内容块、健康建议内容块、检查信息表格都不能突破内容区边界。
4. 健康建议背景块不要贴到弹框左右边缘。
5. 所有内容块左右保留统一边距，建议弹框内容区左右 padding 为 24px。
6. 去掉健康建议相关样式中的负 margin、100vw、calc(100% + xx) 等导致横向溢出的写法。

具体要求：

一、统一内容区 padding

请将弹框滚动内容区设置为：

.sheet-scroll-body {
  flex: 1 1 auto;
  overflow-y: auto;
  padding: 20px 24px 28px;
  box-sizing: border-box;
}

二、统一模块宽度

所有 detail-block 统一：

.detail-block {
  width: 100%;
  box-sizing: border-box;
  margin-bottom: 20px;
}

三、当前结论内容

当前结论正文不需要背景块，但必须和其他内容块左侧对齐：

.plain-content {
  width: 100%;
  box-sizing: border-box;
  font-size: 15px;
  line-height: 1.65;
  font-weight: 700;
  color: #17313a;
}

四、异常指标内容块

异常指标内容块必须位于内容区内，不允许贴边，不允许溢出：

.abnormal-block .soft-content {
  width: 100%;
  box-sizing: border-box;
  padding: 12px 14px;
  border-radius: 12px;
  background: rgba(255, 255, 255, 0.76);
  border: none;
  color: #17313a;
  font-size: 15px;
  font-weight: 700;
  line-height: 1.55;
}

五、健康建议内容块

健康建议内容块必须与当前结论正文、异常指标内容块左侧对齐。
健康建议浅青色背景不能贴到弹框边缘。

.advice-block .advice-content {
  width: 100%;
  box-sizing: border-box;
  padding: 14px 16px;
  border-radius: 14px;
  background: rgba(225, 250, 247, 0.92);
  border: none;
  color: #00a9a5;
  font-size: 15px;
  font-weight: 700;
  line-height: 1.6;
}

六、检查信息表格

检查信息表格也必须遵守同一内容区边界：

.info-grid {
  width: 100%;
  box-sizing: border-box;
}

七、重点检查并删除以下可能导致错位的样式

请检查并删除或覆盖以下样式：
1. .advice-block 的 margin-left: -xx;
2. .advice-block 的 margin-right: -xx;
3. .advice-content 的 margin-left: -xx;
4. .advice-content 的 margin-right: -xx;
5. .advice-content 的 width: 100vw;
6. .advice-content 的 width: calc(100% + xx);
7. .sheet-scroll-body 内子元素的横向溢出样式；
8. 任何导致健康建议背景块超出内容区 padding 的样式。

八、视觉结果要求

调整后视觉应表现为：
1. 当前结论标题、异常指标标题、健康建议标题、检查信息标题左侧对齐。
2. 当前结论正文、异常指标白色背景块、健康建议浅青色背景块、检查信息表格左侧对齐。
3. 异常指标和健康建议背景块距离弹框左右边缘都有明显边距。
4. 健康建议不再像横向色带，而像内容区内的一张浅色提示卡。
5. 不影响弹框固定高度和内部滚动。
