请优化家长端移动端“问卷量表”页面的问卷列表卡片样式，不调整页面整体结构、顶部导航和底部 Tab，只优化列表卡片内容展示。

优化要求：

1. 精简问卷卡片中的信息字段
   - 删除每条问卷卡片第二行中的“健康问卷 / 家长问卷 / 心理量表”等分类字段。
   - 删除每条问卷卡片中的学生姓名“林一凡”。
   - 每条卡片只保留问卷标题、截止日期、状态标签和操作按钮。

2. 截止日期展示
   - 保留截止日期信息。
   - 建议统一展示为：
     “截止日期：2026-04-10”
   - 截止日期放在标题下方，字号小于标题，颜色使用灰蓝色或次级文本色。
   - 避免换行过于零散，尽量保持一行展示。

3. 状态标签优化
   - 当前“未填写 / 填写中 / 已提交”状态标签尺寸过大，请改为小号胶囊标签。
   - 标签高度控制在 20px 左右。
   - 字号控制在 11px-12px。
   - 左右内边距控制在 8px 左右。
   - 圆角使用 10px 左右。
   - 标签不再使用大圆形或大色块，降低视觉占用。

4. 不同状态样式建议
   - 未填写：浅紫色背景 + 紫色文字。
   - 填写中：浅橙色背景 + 橙色文字。
   - 已提交：浅绿色背景 + 绿色文字。
   - 已过期如有：浅灰色背景 + 灰色文字。

5. 卡片布局调整
   - 每条卡片采用紧凑的左右布局：
     左侧：问卷标题 + 截止日期；
     右侧：状态小标签 + 操作按钮。
   - 状态标签与操作按钮横向排列，保持垂直居中。
   - 标题区域占据主要宽度，避免右侧元素挤压导致标题过早换行。
   - 卡片整体高度可适当降低，使列表更紧凑。

6. 操作按钮优化
   - “立即填写”“继续填写”“查看结果”等按钮保留。
   - 按钮高度建议控制在 28px-30px。
   - 字号 12px-13px。
   - 按钮使用浅青绿色背景或描边风格，与当前页面风格保持一致。
   - 不要让按钮高度明显大于状态标签太多。

7. 视觉一致性
   - 保持当前页面的浅绿色背景、白色卡片、圆角风格。
   - 卡片圆角、阴影、间距不要大幅改变。
   - 重点是减少重复字段、压缩状态标签、提升信息阅读效率。

目标效果：
问卷列表更像一个清晰的“待办问卷列表”，用户能快速看到：
问卷名称、截止日期、当前状态、可执行操作。
页面整体要更简洁、紧凑、移动端友好。
