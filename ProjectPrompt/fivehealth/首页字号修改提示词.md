继续优化五健家长端首页的文字层级和模块标题。

本次只调整首页快捷入口文字、健康宣教模块标题和整体文字权重，不修改页面结构、不修改业务逻辑、不修改底部导航。

当前问题：
1. 快捷入口中的“报告查询、体检报名、问卷量表、复筛通知”文字字重偏重，和图标一起显得过于抢眼。
2. 页面中多个标题和入口文字都比较粗，导致视觉重点过多。
3. “健康宣教”作为家长端模块标题略偏后台化，且字重较重，影响页面清爽感。

修改目标：
1. 降低快捷入口文字权重。
2. 降低页面中非核心标题的视觉强度。
3. 将“健康宣教”改为更适合家长端的“健康知识”，或者至少弱化其样式。
4. 保留“更多”入口。
5. 页面整体更轻、更清爽，但核心报告信息仍然突出。

具体要求：

一、快捷入口文字调整

快捷入口包括：
- 报告查询
- 体检报名
- 问卷量表
- 复筛通知

调整要求：
1. 字号建议 13px。
2. 字重建议 font-weight: 500。
3. 颜色不要使用纯黑，建议使用 #1f3b43 或 #425c64。
4. 不要使用 700 或 bold。
5. 图标保持不变。
6. 快捷入口仍需清晰可读，但不能比“体检报告卡”更抢眼。

参考样式：

.quick-entry-title {
  font-size: 13px;
  font-weight: 500;
  color: #425c64;
  line-height: 1.3;
}

二、健康宣教标题调整

将模块标题“健康宣教”改为“健康知识”。

标题区域保留右侧“更多”。

调整后展示：
健康知识                                      更多

样式要求：
1. 标题字号建议 16px。
2. 字重建议 600。
3. 颜色建议 #1f3b43。
4. 不要使用过粗字重。
5. “更多”保持主色青绿色，但字号不要过大。

参考样式：

.section-title {
  font-size: 16px;
  font-weight: 600;
  color: #1f3b43;
}

.section-more {
  font-size: 13px;
  font-weight: 500;
  color: #00a9a5;
}

三、健康文章标题调整

健康文章标题如：
孩子视力下降后，家长应该怎么做

调整要求：
1. 字号建议 15px。
2. 字重建议 600。
3. 不要使用 700 或 bold。
4. 保持两行省略逻辑不变。

四、核心报告卡保持突出

以下内容仍然可以保持较强权重：
1. 当前学生姓名与班级。
2. 最近体检报告标题。
3. “存在 3 项需关注指标”中的数字 3。

不要削弱这些核心信息。

五、禁止事项

1. 不要修改页面布局结构。
2. 不要删除健康知识模块。
3. 不要删除分类标签。
4. 不要修改快捷入口顺序。
5. 不要修改图片和图标资源。
6. 不要修改业务跳转逻辑。
