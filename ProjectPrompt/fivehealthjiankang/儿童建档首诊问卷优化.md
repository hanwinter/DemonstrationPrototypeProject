请继续优化当前家长端 H5 第二版“五健智行/专案管理”中的两个二级页面：

1. 儿童建档页
2. 首诊问卷页

本次重点是字段拆分与页面结构优化：
- 儿童建档页只放“基础身份信息、就读信息、监护人联络信息、出生信息、既往健康与家族风险史”等建档必填/长期稳定信息；
- 首诊问卷页放“生活行为基线、用眼习惯、饮食睡眠、运动、口腔卫生、体态习惯、专项风险线索”等需要问卷采集、可能随时间变化的信息；
- 五大专项筛查核心指标，如身高体重、BMI、视力、屈光、眼轴、龋齿、ATR角度、心理量表结果等，不要放进儿童建档页和首诊问卷页的主体录入区，后续应归入专项档案/筛查结果/报告页。本次只在建档页中保留“五健专项档案概览”入口或状态展示即可。

一、儿童建档页优化

页面标题：儿童建档

页面定位：
用于完成儿童基础建档，采集长期稳定的身份、就读、联络、出生及健康风险信息。页面不要做成长问卷，也不要录入五大专项筛查结果。

保留当前页面整体风格：
- 浅青绿色背景；
- 顶部返回栏；
- 学生/专案摘要卡片；
- 白色分组卡片；
- 小圆角、轻阴影、弱边框；
- 底部固定主按钮。

页面结构建议：

1. 学生/专案摘要卡片
展示：
- 林一凡｜男｜10岁
- 视力健康管理专案
- 当前阶段：儿童建档

2. 基础身份信息
字段：
- 儿童姓名
- 性别
- 出生日期
- 年龄（可展示为自动计算结果）
- 身份证号
- 儿童健康档案编号

说明：
- 年龄不要作为唯一录入字段，建议由出生日期计算或以只读展示形式呈现。
- 身份证号和健康档案编号可先用 mock 数据。

3. 就读信息
字段：
- 就读机构：幼儿园 / 学校
- 年级
- 班级
- 学籍号

4. 监护人及联络信息
字段：
- 监护人姓名
- 与儿童关系
- 联系电话
- 紧急联系人姓名
- 紧急联系人电话
- 户籍地址
- 现居住地址

说明：
- 至少突出一个紧急联系人信息。
- 地址字段可用多行输入框样式或紧凑文本域样式。

5. 出生与成长基础信息
字段：
- 民族
- 出生孕周
- 是否早产
- 分娩方式

说明：
- 是否早产、分娩方式建议使用轻量单选/标签选择样式。
- 不要占用过多空间。

6. 既往健康与家族风险史
字段：
- 既往病史
  - 哮喘
  - 过敏
  - 矮小
  - 肥胖
  - 弱视
  - 脊柱疾病
  - 精神情绪相关病史
  - 无明显既往病史
- 手术史
- 长期用药史
- 家族高危史
  - 父母近视
  - 父母高度近视
  - 家族肥胖
  - 糖尿病
  - 脊柱侧弯
  - 心理疾病遗传史
  - 无明显家族高危史
- 既往体检异常记录
- 以往就诊 / 矫正记录

UI要求：
- 既往病史和家族高危史建议用“标签多选”样式，不要全部做成长文本输入。
- 手术史、长期用药史、既往异常记录、就诊矫正记录可以用简短文本域。
- 如果选择“无明显既往病史”或“无明显家族高危史”，要在视觉上与其他标签互斥，逻辑可先不做真实校验，样式要预留。

7. 关联任务入口
在页面底部、提交按钮上方增加两个轻量入口卡片：
- 生活行为基线问卷：未填写，点击进入首诊问卷
- 五健专项档案概览：待筛查，点击可先显示 toast 或跳转占位页

说明：
- 这两个入口要轻量展示，不要做成大卡片。
- 用于说明这些信息不在儿童建档页里直接填写。

8. 底部主按钮
按钮文案：
- 提交建档信息

点击后：
- 前端模拟 toast：建档信息已保存
- 不需要真实接口。

二、首诊问卷页优化

页面标题：首诊问卷

页面定位：
用于采集生活行为基线、视力相关用眼习惯、运动饮食睡眠、口腔卫生、体态习惯和专项风险线索。该页面是问卷，不是基础建档页，也不是筛查结果录入页。

保留当前页面整体风格：
- 与儿童建档页一致；
- 顶部返回栏；
- 学生/专案摘要卡片；
- 问卷题目卡片；
- 底部固定提交按钮。

页面结构建议：

1. 学生/专案摘要卡片
展示：
- 林一凡｜男｜10岁
- 视力健康管理专案
- 当前阶段：首诊问卷

2. 问卷进度
展示：
- 问卷进度 3/18
或
- 已完成 3/18

UI要求：
- 进度展示轻量即可；
- 可以使用细进度条或文本 + 小进度条；
- 不要做成很重的大进度条。

3. 问卷分组

建议分为以下几个分组，每组用轻量标题，不要每组都做很重的大卡片。

分组一：用眼与视力行为
题目示例：
- 每日近距离用眼时长
  选项：少于1小时 / 1-2小时 / 2-4小时 / 4小时以上
- 每日电子产品使用时长
  选项：少于30分钟 / 30分钟-1小时 / 1-2小时 / 2小时以上
- 是否经常眯眼看远处
  选项：从不 / 偶尔 / 经常
- 是否频繁揉眼、眼干或眼疲劳
  选项：无 / 偶尔 / 经常
- 是否已经佩戴眼镜
  选项：未佩戴 / 已佩戴框架眼镜 / 已佩戴角膜塑形镜
- 初发近视时间
  输入或选择：未近视 / 1年内 / 1-3年 / 3年以上

分组二：户外活动与运动
题目示例：
- 每日户外活动时长
  选项：少于30分钟 / 30分钟-1小时 / 1-2小时 / 2小时以上
- 每周运动频次
  选项：少于1次 / 1-2次 / 3-4次 / 5次及以上
- 主要运动类型
  选项：跑跳类 / 球类 / 游泳 / 舞蹈体操 / 其他

分组三：饮食与睡眠
题目示例：
- 三餐是否规律
  选项：规律 / 偶尔不规律 / 经常不规律
- 是否经常摄入甜食或含糖饮料
  选项：很少 / 每周1-2次 / 每周3次以上 / 几乎每天
- 是否每天吃早餐
  选项：每天 / 偶尔不吃 / 经常不吃
- 每日睡眠时长
  选项：少于7小时 / 7-8小时 / 8-9小时 / 9小时以上
- 作息是否规律
  选项：规律 / 偶尔晚睡 / 经常晚睡

分组四：口腔卫生习惯
题目示例：
- 每日刷牙次数
  选项：0次 / 1次 / 2次 / 2次以上
- 是否使用牙线
  选项：从不 / 偶尔 / 经常
- 定期口腔检查频率
  选项：从不 / 每年1次 / 每半年1次 / 有问题才检查
- 是否做过窝沟封闭或涂氟
  选项：未做过 / 做过窝沟封闭 / 做过涂氟 / 不清楚

分组五：体态与脊柱习惯
题目示例：
- 日常坐姿情况
  选项：较好 / 偶尔歪斜 / 经常歪斜
- 书包使用情况
  选项：双肩书包 / 单肩书包 / 经常单侧背包
- 是否有高低肩、驼背、走路姿态异常等情况
  选项：无 / 不确定 / 有
- 是否经常久坐
  选项：很少 / 偶尔 / 经常

分组六：心理与情绪线索
题目示例：
- 最近是否明显情绪低落或烦躁
  选项：无 / 偶尔 / 经常
- 睡眠质量是否较差
  选项：无 / 偶尔 / 经常
- 是否存在明显厌学、焦虑或人际交往困难
  选项：无 / 不确定 / 有
- 是否有注意力不集中或多动冲动表现
  选项：无 / 偶尔 / 经常

4. UI要求
- 问卷题目使用“题目卡片”或紧凑题目块；
- 单选、多选按钮使用浅色描边标签样式；
- 不要使用过重的蓝绿色大色块；
- 每个分组之间要有清晰层级；
- 页面需要适配 390px 移动端宽度；
- 避免内容横向溢出；
- 底部固定按钮不能遮挡最后一道题，内容底部要预留安全距离。

5. 底部主按钮
按钮文案：
- 提交首诊问卷

点击后：
- 前端模拟 toast：首诊问卷已提交
- 不需要真实接口。

三、不要做的内容

1. 不要把五大专项筛查核心指标直接放进儿童建档页：
   - 身高、体重、BMI
   - 左右眼视力
   - 球镜、柱镜、轴位、远视储备
   - 眼轴长度
   - 龋齿颗数、窝沟封闭、涂氟记录
   - ATR角度、高低肩、扁平足、X/O型腿
   - 心理量表测评结果、心理风险等级

2. 不要把首诊问卷做成筛查结果录入页。
3. 不要一次性展开过多字段导致页面非常长且难以填写。
4. 不要改变当前专案流程页、报告页、家庭服务页等已完成页面的整体结构。
5. 不要影响其他底部 Tab 和路由。

四、交互要求

1. 儿童建档页底部“生活行为基线问卷”入口点击后进入首诊问卷页。
2. 儿童建档页底部“五健专项档案概览”入口可先 toast 提示：专项档案将在筛查完成后生成。
3. 儿童建档页提交按钮点击后 toast：建档信息已保存。
4. 首诊问卷页提交按钮点击后 toast：首诊问卷已提交。
5. 返回按钮均返回当前专案流程页。

五、验收重点

请重点检查：
1. 儿童建档页和首诊问卷页字段边界是否清楚；
2. 儿童建档页是否只保留基础建档和长期稳定信息；
3. 首诊问卷页是否承接生活行为和专项风险线索；
4. 五大专项筛查指标是否没有被错误放入建档/问卷主体；
5. UI风格是否与当前家长端 H5 二级页一致；
6. 移动端 390px 宽度下是否无横向滚动、无内容挤压；
7. 底部按钮是否不遮挡内容。
