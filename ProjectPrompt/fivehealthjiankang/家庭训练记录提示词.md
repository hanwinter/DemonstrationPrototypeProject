请继续优化当前家长端 H5 第二版“五健智行/专案管理”中的“家庭训练计划”二级页面。

本次重点：参照“复诊计划”页面的优化方向，将家庭训练计划页面从“多个独立任务卡片堆叠”优化为更适合家长端执行和记录的“训练计划通知单 + 任务清单”样式。保持当前页面整体风格一致，不做大幅重构。

一、当前问题

当前家庭训练计划页面存在以下问题：
1. 周期字段和值字号差距较大，视觉不协调；
2. 训练任务分别用多张独立白卡片展示，页面纵向占用较多；
3. 每个任务右侧“未完成/已完成”状态顶到最右侧，阅读路径有些割裂；
4. 任务说明和状态之间缺少整体任务清单感；
5. 家长记录区域可以保留，但整体需要更紧凑、层级更清楚。

二、优化目标

请将页面优化为：

1. 顶部学生/专案摘要卡片保持不变；
2. 周期信息改为更像“训练计划周期”的轻量通知条；
3. 训练任务合并到一张“本周训练任务”卡片中；
4. 每个任务在同一张卡片内用紧凑行展示；
5. 任务之间用极浅色分割线；
6. 任务状态使用无边框小标签；
7. 家长记录区域保持独立卡片；
8. 底部按钮“提交训练记录”保持当前风格。

三、周期信息优化

当前：
- 周期字段和值紧贴，字号差距明显。

调整要求：
1. 将周期信息展示为一张轻量通知条或小卡片。
2. 推荐布局：
   - 第一行：训练周期
   - 第二行：2026-04-15 至 2026-04-21    待提交
3. 字号建议：
   - 字段名：12-13px；
   - 周期值：15-16px；
   - 不要让周期值像大标题一样突兀。
4. 状态“待提交”放在周期值同一行右侧，使用小号橙色标签。
5. 卡片整体保持轻量，不要使用大面积色块。

四、训练任务区域优化

当前：
- 户外活动、远眺训练、用眼间隔、眼保健操分别是独立卡片，页面松散。

调整要求：
1. 将所有训练任务合并到一张卡片中。
2. 卡片标题：
   - 本周训练任务
3. 任务列表：
   - 户外活动：每日不少于 2 小时，优先选择自然光环境。
   - 远眺训练：每日 3 次，每次 5 分钟。
   - 用眼间隔：近距离用眼 30 分钟后休息 10 分钟。
   - 眼保健操：每日 1 次，注意动作规范。
4. 每个任务展示结构：
   - 第一行：任务名称 + 状态标签
   - 第二行：任务说明
5. 状态标签：
   - 未完成：浅橙色小标签；
   - 已完成：浅青绿色小标签；
   - 无边框；
   - 小圆角；
   - 不要顶到特别远导致阅读断开；
   - 可以放在任务名称右侧，保持近距离。
6. 任务之间使用极浅色分割线；
7. 最后一项任务底部不需要分割线。
8. 不要给每个任务再套独立白卡片。

五、任务状态标签样式

1. 去掉任务状态标签的边框。
2. 使用浅底色小标签：
   - 已完成：浅青底 + 青绿色文字；
   - 未完成：浅橙底 + 橙色文字。
3. 标签字号小一些；
4. 标签高度不要过高；
5. 不要让状态标签像按钮。

六、家长记录区域优化

当前家长记录区域可保留，但需要保持紧凑。

保留内容：
- 今日已完成
- 今日未完成
- 训练备注输入框
- 上传图片入口

调整要求：
1. 卡片标题：家长记录
2. “今日已完成 / 今日未完成”保持轻量分段按钮样式；
3. 输入框保持当前浅色边框样式；
4. 上传图片入口保持虚线框，但高度不要过高；
5. 家长记录卡片与训练任务卡片间距缩小，保持页面紧凑；
6. 不要新增真实上传逻辑。

七、页面整体风格要求

1. 保持当前家长端 H5 风格：
   - 浅青绿色背景；
   - 白色卡片；
   - 小圆角；
   - 轻阴影或弱边框；
   - 青绿色主色；
   - 橙色仅用于未完成/待提交状态。
2. 页面应像“家庭训练计划通知单 + 执行记录”，不要像多个散开的任务按钮。
3. 不要改变顶部栏、学生摘要卡片、底部固定按钮的整体风格。
4. 页面内容底部要预留安全距离，避免底部按钮遮挡。

八、不要做的内容

1. 不要修改其他二级页面；
2. 不要修改专案页面；

4. 不要把每个训练任务继续做成独立卡片；
5. 不要让状态标签使用边框样式；
6. 不要让状态标签像可点击按钮；
7. 不要大幅重构页面视觉。

1. 周期字段和值字号是否更协调；
2. 周期状态“待提交”是否清晰但不抢眼；
