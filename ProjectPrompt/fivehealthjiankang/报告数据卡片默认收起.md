请继续优化当前“五健健康管理家长端 H5”中的“专科诊疗报告”二级页面。

本次重点：报告详情中的各个数据/内容分组默认全部收起，只保留顶部报告摘要卡片默认展示。

一、当前问题

当前进入“专科诊疗报告”二级页后，“核心检测数据”等分组默认展开，表格内容较多，一屏内占用很大空间，用户难以先看到报告整体结构。

二、调整目标

页面打开后：
1. 顶部报告摘要卡片默认展示；
2. 下方所有报告分组默认收起；
3. 用户点击“展开”后再查看具体内容；
4. 展开后可点击“收起”恢复。

三、默认收起的分组

请将以下所有分组默认设置为收起状态：

1. 核心检测数据
2. 诊断结论与风险分级
3. 个性化干预方案
4. 随访、复查与专案管理记录
5. 基础建档信息
6. 主诉、现病史、高危家族史
7. 眼部常规专科查体
8. 原始检查单据归档
9. 附件

注意：
- 不要默认展开“核心检测数据”；
- 不要默认展开诊断、干预、复查等内容；
- 只保留顶部报告摘要卡片始终展示。

四、收起状态 UI 要求

每个分组收起时展示：
1. 分组标题；
2. 右侧“展开”；
3. 可选一行极简摘要，不强制。

示例：
- 核心检测数据          展开
- 诊断结论与风险分级    展开
- 个性化干预方案        展开
- 原始检查单据归档      展开

UI要求：
1. 收起卡片高度保持紧凑；
2. 标题左侧短青色竖线保留；
3. 不要留出内容区域空白；
4. 卡片之间间距自然；
5. 与当前页面白色卡片、小圆角、浅青背景风格一致。

五、展开状态 UI 要求

用户点击某个分组“展开”后：
1. 当前分组展示完整内容；
2. 右侧按钮变为“收起”；
3. 展开内容保持现有样式；
4. 表格、文本、LIS/PACS 列表等内容不需要重做；
5. 再次点击“收起”后隐藏内容。

六、交互方式

1. 各分组可以独立展开/收起；
2. 可以同时展开多个分组；
3. 不需要实现“全部展开/全部收起”；
4. 不需要接真实接口；
5. 使用前端状态控制即可。

七、不要做的内容

1. 不要修改顶部报告摘要卡片；
2. 不要删除任何报告内容；
3. 不要修改 mock 数据；
4. 不要修改专案进展页面；
5. 不要修改康复、家庭服务、AI自测等页面；
6. 不要改变二级页整体 UI 风格；
7. 不要新增复杂动画。

八、验收重点

请检查：
1. 进入专科诊疗报告页后，是否只有顶部摘要卡片展开显示；
2. 核心检测数据是否默认收起；
3. 所有报告分组是否默认收起；
4. 点击展开后是否能正常显示内容；
5. 点击收起后是否能隐藏内容；
6. 收起状态下是否没有异常空白；
7. 390px 移动端宽度下是否无横向滚动、无内容挤压。
