请优化“五健健康管理家长端 H5”模块中的“图文报告 / 查看报告”二级页面。

特别注意：
1. 本次只调整“五健健康管理家长端 H5”模块。
2. 不要修改当前已经定版的“家长端 H5 演示”模块。
3. 不要修改现有“五健智行专案家长端 H5”模块。
4. 不要影响“五健健康管理家长端 H5”中的首页、专案、报告、我的等其他页面逻辑。
5. 本次重点优化 LIS/PACS 图文报告查看页面的顶部返回样式和报告查看工具栏。

当前问题：
1. 图文报告页面顶部返回按钮样式不统一。
2. 页面卡片内部又出现了一个“返回”按钮，造成返回入口重复。
3. 报告图片被压缩展示，缺少放大、缩小、适应屏幕能力。
4. 页面还不像一个真正的报告查看器，更像普通卡片里放了一张图片。

一、顶部返回样式统一
请将图文报告二级页顶部栏调整为与“五健健康管理家长端 H5”中其他二级页一致的样式，例如问卷、查看报告等二级页。

要求：
1. 顶部左侧使用统一返回按钮样式。
2. 返回按钮位置、尺寸、颜色、圆角、图标风格与其他二级页保持一致。
3. 页面主体卡片内部不要再放“返回”按钮。
4. 整个页面只保留一个明确的返回入口。
5. 顶部标题建议使用：
   - 图文报告
   或
   - 查看报告
6. 点击顶部返回按钮应返回专案报告页或上一页。

推荐顶部结构：
- 左侧：返回图标
- 中间：图文报告 / 查看报告
- 右侧：如项目二级页已有统一占位或图标，按项目规范处理；否则可以留空

二、报告基础信息展示
报告名称和检查日期不要与返回按钮挤在一起。

请在顶部栏下方展示报告基础信息：
1. 报告名称，例如：维生素D检测
2. 报告类型，例如：LIS 化验单
3. 检查日期，例如：2026-04-08

推荐展示：
- 维生素D检测
- LIS 化验单｜2026-04-08

PACS 示例：
- 超声报告单
- PACS 检查报告｜2026-04-08

三、增加报告查看工具栏
在报告基础信息下方、报告图片上方增加工具栏。

工具栏包含 3 个按钮：
1. 缩小
2. 放大
3. 适应屏幕

要求：
1. 三个按钮必须显示在页面中。
2. 按钮样式轻量，不要做成强主按钮。
3. 推荐使用浅绿色背景、白底描边或轻量胶囊按钮。
4. 字号不要过大。
5. 按钮之间间距均匀。
6. 工具栏不要放在底部，避免被 H5 容器或系统导航遮挡。
7. 工具栏建议放在报告信息下方、图片查看区上方。

推荐结构：
- [缩小] [放大] [适应屏幕]

如果项目已有图标库，也可以使用：
- - 
- +
- 适应屏幕
但文字按钮优先，便于演示理解。

四、图片缩放逻辑
请为报告图片增加基础缩放能力。

要求：
1. 初始状态：图片适应屏幕宽度。
2. 点击“放大”：图片按比例放大。
3. 点击“缩小”：图片按比例缩小。
4. 点击“适应屏幕”：恢复到适应容器宽度。
5. 缩小不能无限缩小，最低恢复到适应屏幕状态。
6. 放大后支持拖动或滚动查看局部内容。
7. 如果实现拖动成本较高，至少保证放大后可以通过横向/纵向滚动查看。
8. 图片保持原始比例，不要裁剪。
9. 图片不要被强行压缩到不可读。

建议缩放规则：
1. 默认 scale = 1
2. 放大每次 +0.2 或 +0.25
3. 缩小每次 -0.2 或 -0.25
4. 最小 scale = 1
5. 最大 scale 可控制在 2.5 或 3
6. 适应屏幕重置 scale = 1

五、报告图片查看区
请将报告查看区域优化为阅读器样式。

要求：
1. 报告图片区域尽量大。
2. 图片查看区背景可以使用浅灰或白色。
3. 报告图片居中展示。
4. 图片放大后不要被外层卡片裁剪得无法查看。
5. 图片区域允许滚动。
6. 如果图片超出容器宽度，允许横向滚动。
7. 查看报告时不要显示底部 Tab，避免占用阅读空间。
8. 外层卡片不要使用过大的圆角和过重边框。
9. 页面整体仍保持五健健康管理 H5 的清爽、绿色/蓝绿色风格。

六、页面结构建议
优化后页面结构建议为：

1. 顶部栏
   - 返回按钮
   - 标题：图文报告

2. 报告信息区
   - 维生素D检测
   - LIS 化验单｜2026-04-08

3. 工具栏
   - 缩小
   - 放大
   - 适应屏幕

4. 报告查看区
   - 展示报告图片
   - 支持缩放与滚动/拖动查看

要求：
1. 点击 LIS 单据的“图文报告”后，进入该查看器并展示 LIS 图片。
2. 点击 PACS 报告的“图文报告”后，进入该查看器并展示 PACS 图片。
3. LIS 图片使用：

   - /ProjectImage/超声报告单.png
   - 或通过 import 引入资源
6. 页面中不要出现 PASS 字样。
