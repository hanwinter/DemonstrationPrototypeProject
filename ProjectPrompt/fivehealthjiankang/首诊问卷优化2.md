请继续优化当前家长端 H5 第二版“五健智行/专案管理”中的“首诊问卷”二级页面。

本次重点：调整问卷页面视觉层级和页面密度，解决“分组标题与问题标题视觉效果接近”“卡片间距偏大”的问题。

一、当前问题

从当前页面看：
1. 分组标题如“户外活动与运动”和问题标题如“每日户外活动时长”字号、字重、颜色接近，视觉层级不明显，容易误读。
2. 各个问卷分组卡片之间间距偏大，尤其折叠状态下像一组很松散的大按钮。
3. 展开卡片内部问题块上下留白略大，页面纵向空间利用率不够高。

二、优化目标

请在不改变问卷字段和交互逻辑的前提下，优化：
- 分组标题层级；
- 问题标题层级；
- 问题块间距；
- 分组卡片间距；
- 折叠卡片高度。

整体目标：
让页面看起来更像一份紧凑、专业、层级清晰的健康问卷。

三、分组标题优化

1. 分组标题要明显高于问题标题，但不要做成大色块。
2. 推荐样式：
   - 分组标题左侧增加一条短青色竖线；
   - 标题字号比问题标题大 1-2px；
   - 字重更高；
   - 颜色使用深色标题色；
   - 右侧“展开 / 收起”保持小号青绿色文字。
3. 示例：
   - 左侧短竖线 + 户外活动与运动
   - 右侧：收起
4. 短竖线要求：
   - 高度约 14-16px；
   - 宽度约 3px；
   - 圆角；
   - 颜色使用当前系统主青绿色；
   - 与标题文字间距 6px 左右。
5. 分组标题行高度不要过高，保持紧凑。

四、问题标题优化

1. 问题标题视觉层级要低于分组标题。
2. 调整要求：
   - 字号略小于分组标题；
   - 字重从 bold 降为 medium 或 500；
   - 颜色使用深灰，不要过黑；
   - 不要加左侧竖线；
   - 不要和分组标题同样粗重。
3. 问题标题与选项之间的间距控制在 6-8px。
4. 问题标题本身保持清晰，但不要抢分组标题层级。

五、问题块间距与分割线

1. 保留问题与问题之间的极浅色分割线。
2. 分割线颜色继续使用很浅的颜色，例如 #EEF5F4 或当前系统同类浅线色。
3. 每个问题块上下 padding 缩小：
   - 问题块上方间距约 10-12px；
   - 问题块下方间距约 10-12px。
4. 不要依靠大留白区分题目，主要用浅色分割线建立秩序。
5. 最后一题底部不需要分割线。
6. 选项行之间如果发生换行，行距要自然，不要拥挤。

六、分组卡片间距优化

1. 问卷进度卡片与第一个问卷分组之间间距控制在约 10px。
2. 各问卷分组卡片之间间距控制在约 8-10px。
3. 不要再使用过大的 14-16px 间距。
4. 折叠状态下的分组卡片不要显得像大按钮。

七、折叠状态卡片优化

1. 折叠状态只显示标题行和右侧“展开”。
2. 折叠卡片高度控制在约 46-50px。
3. 标题垂直居中。
4. 不要在折叠状态下保留额外空白。
5. 如果显示完成数，例如“3/6”，要弱化，不能抢标题。

八、展开状态卡片优化

1. 展开状态标题行下方进入问题内容。
2. 标题区和第一题之间留 10-12px 即可。
3. 卡片内边距保持统一，建议左右 14-16px。
4. 展开卡片不要过度留白，但也不能挤压题目和选项。
5. 保持整体轻量白卡片风格。

九、保持不变的内容

1. 不要改变问卷字段内容；
2. 不要改变“保存草稿 / 提交首诊问卷 / 已提交只读”的状态逻辑；
3. 不要改变分组折叠/展开逻辑；
4. 不要把每道题重新做成独立卡片；
5. 不要影响儿童建档页和其他二级页面；
6. 不要新增真实接口或复杂校验。

十、验收重点

请重点检查：
1. 分组标题是否明显高于问题标题；
2. 分组标题左侧是否有短青色竖线；
3. 问题标题是否降级，不再与分组标题视觉相同；
4. 问题之间是否用浅色分割线区分；
5. 问题块上下间距是否缩小；
6. 分组卡片之间间距是否缩短；
7. 折叠卡片高度是否更紧凑；
8. 390px 移动端宽度下是否无横向滚动、无内容重叠；
9. 底部按钮是否不遮挡最后一道题。
