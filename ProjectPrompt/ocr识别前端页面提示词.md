# 项目名称

# 页面名称

门诊高拍仪识别

# 页面标题

外院报告识别校验

# 技术要求

使用以下技术栈：

* Vue3
* TypeScript
* Element Plus
* Composition API
* 单文件组件（SFC）
* 使用 Mock 数据
* 不连接后端
* 所有操作均为前端模拟
* 页面用于医院项目演示

要求代码结构清晰，可直接运行。

---

# 页面整体布局

采用左右布局。

右侧宽度约 65%

整体风格：

* 医疗行业业务系统
* 浅色背景

* Element Plus 风格
* 不要互联网产品风格
* 不要卡片堆砌
* 保持医院信息系统风格
