当前问题：
登录页背景太浅，接近纯白，没有形成参考图中那种青绿色移动端登录页的主视觉效果。现在登录卡片像漂浮在浅白背景上，缺少儿童健康管理平台的品牌感。

本次重点调整登录页底色和背景结构。

一、登录页背景结构调整

请将登录页背景改为上下分区视觉：

1. 上半部分使用大面积青绿色渐变背景
2. 下半部分使用浅薄荷绿或近白背景
3. 登录卡片放在两种背景交界处，形成白色卡片压在青绿色背景上的效果

参考结构：

- 顶部 42% 到 48% 高度：青绿色渐变背景
- 底部剩余区域：浅薄荷绿背景
- 登录卡片：白色圆角卡片，向上覆盖青绿色区域下沿

二、具体颜色要求

顶部主视觉背景使用：

background: linear-gradient(180deg, #46D2D2 0%, #26BFC3 45%, #12A8AD 100%);

或者：

background: linear-gradient(135deg, #46D2D2 0%, #12A8AD 100%);

底部背景使用：

background: #EAF8F5;

或者：

background: linear-gradient(180deg, #EAF8F5 0%, #F8FEFC 100%);

不要继续使用接近纯白的整页背景。

三、页面容器实现建议

可以在登录页根容器中增加背景层：

.login-page {
  min-height: 100vh;
  background: #EAF8F5;
  position: relative;
  overflow: hidden;
}

.login-page::before {
  content: "";
  position: absolute;
  left: 0;
  top: 0;
  width: 100%;
  height: 46%;
  background: linear-gradient(135deg, #46D2D2 0%, #12A8AD 100%);
  border-bottom-left-radius: 36px;
  border-bottom-right-radius: 36px;
  z-index: 0;
}

登录页内容区域需要设置 position: relative; z-index: 1;

四、顶部区域视觉

在青绿色背景区域内展示：

1. 平台标题：
儿童健康管理平台

2. 主视觉文案：
关注孩子成长健康

3. 副标题：
筛查报告、复筛提醒、健康建议一站查看

文字颜色统一使用白色或半透明白色：

标题：#FFFFFF
副标题：rgba(255,255,255,0.86)

五、背景点缀

请在顶部青绿色背景中增加轻量点缀：

1. 半透明圆点
2. 浅色健康十字
3. 柔和波纹
4. 五健小图标淡化装饰

点缀颜色使用：

rgba(255, 255, 255, 0.16)
rgba(255, 255, 255, 0.24)

不要过多，不要影响文字阅读。

六、登录卡片位置

登录卡片需要整体上移，压在青绿色背景下沿附近。

建议：

.login-card {
  margin: -40px 24px 0;
  background: #FFFFFF;
  border-radius: 24px;
  box-shadow: 0 18px 44px rgba(18, 168, 173, 0.20);
}

如果当前结构不方便使用负 margin，可以通过 absolute / padding-top 调整，但最终视觉要实现：
登录卡片上半部分靠近青绿色背景，下半部分落在浅薄荷绿背景上。

七、登录卡片颜色保持

登录卡片保持白色，不要改成绿色。

输入框：
background: #FAFEFD;
border: 1px solid #D8EEEA;

获取验证码按钮：
background: #E4F8F6;
color: #12A8AD;

登录按钮：
background: linear-gradient(135deg, #46D2D2 0%, #12A8AD 100%);
color: #FFFFFF;

八、顶部导航栏处理

如果登录页保留顶部导航栏：

1. 导航栏可以透明化，叠在青绿色背景上。
2. 标题使用白色。
3. 返回按钮使用 rgba(255,255,255,0.22) 背景。
4. 返回图标使用白色。
5. 不要在登录页顶部使用白色导航栏，否则会破坏青绿色主视觉。

如果项目统一顶部导航不能透明，则登录页可以单独覆盖样式。

九、不要做的事情

1. 不要把整页背景继续做成接近纯白。
2. 不要只改按钮颜色。
3. 不要把登录卡片改成绿色。
4. 不要使用真实儿童照片。
5. 不要影响其他家长端页面。
6. 不要修改登录逻辑。
7. 不要接真实验证码接口。

十、最终验收标准

完成后登录页应满足：

1. 首屏能明显看到大面积青绿色渐变背景。
2. 登录卡片是白色圆角卡片，压在青绿色背景和浅薄荷绿背景之间。
3. 页面整体更接近移动端 App 登录页，而不是普通后台表单页。
4. 颜色接近参考图的青绿色 / 蓝绿色风格。
5. 仍保留儿童健康管理平台的亲和、清爽、可信赖气质。
