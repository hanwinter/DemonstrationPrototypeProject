一、调整目标

1. 登录页顶部栏不再显示“儿童健康管理平台”标题。
2. 登录页顶部右侧不再显示铃铛或任何通知图标。
3. 登录页顶部只保留左侧返回按钮。
4. 青绿色主视觉背景可以从页面最顶部开始延展。
5. 平台名称统一放在主视觉区中展示。

二、顶部栏调整

请将登录页顶部栏调整为极简模式：

1. 左侧保留返回按钮。
2. 中间标题隐藏，不显示任何文字。
3. 右侧图标隐藏，不显示铃铛，不显示浅色方块。
4. 顶部栏背景透明。
5. 顶部栏不需要底部分割线。
6. 顶部栏只作为返回操作区，不作为品牌标题区。

如果代码中有 page-header-title / nav-title / header-title 等元素，请在登录页中隐藏。

示例样式原则：

.login-page .page-header-title {
  display: none;
}

.login-page .header-right,
.login-page .notification-button,
.login-page .bell-button {
  display: none;
}

.login-page .page-header {
  background: transparent;
  border-bottom: none;
  box-shadow: none;
}

三、返回按钮样式

登录页返回按钮需要清晰可见，但不要太抢眼。

建议样式：

background: rgba(255, 255, 255, 0.22);
color: #FFFFFF;
border: 1px solid rgba(255, 255, 255, 0.28);
backdrop-filter: blur(8px);

如果当前返回按钮在青绿色背景上不清楚，请提高透明度：

background: rgba(255, 255, 255, 0.28);
color: #FFFFFF;

不要使用深色返回按钮。
不要使用浅绿底 + 白色图标的低对比组合。

四、背景区域调整

登录页青绿色主视觉背景可以从页面顶部开始铺开，不需要给顶部栏单独留白。

建议结构：

1. 页面最上方开始就是青绿色渐变背景。
2. 顶部返回按钮浮在青绿色背景上。
3. 主视觉文案位于返回按钮下方。
4. 登录卡片压在青绿色背景下沿附近。
5. 底部仍使用浅薄荷绿背景。
