五、主视觉区设计要求

当前页面顶部空白过多，请在登录卡片上方增加儿童健康主题主视觉区。

主视觉区内容包括：

建议画面元素：
- 一个儿童头像或儿童身体轮廓
- 一个健康档案卡片
- 周围点缀五健图标：
  - 眼睛：视力健康
  - 牙齿：口腔健康
  - 脊柱或身体姿态：脊柱健康
  - 爱心：心理健康
  - 体重秤：体重健康

2. 主视觉背景
使用浅绿色、浅蓝绿色或青绿色渐变背景。
可以增加几处半透明圆点、波纹、十字医疗符号、健康曲线等轻量点缀。

不要使用真实儿童照片。

“关注孩子成长健康”
“筛查报告、复筛提醒、健康建议一站查看”

4. 主视觉高度控制
不要让主视觉过高。
建议控制在登录页上半部分，确保登录卡片在首屏内可见。
整体不要出现当前这种大面积空白。

七、页面视觉风格要求

整体延续儿童健康档案 UI 思路：

1. 风格关键词：
柔和、亲和、可信赖、儿童健康、轻医疗、档案感。

2. 主色：
绿色 / 青绿色。

3. 辅助色：
浅蓝、浅绿、淡橙、淡紫，用于五健图标点缀。

4. 背景：
使用浅绿色渐变背景或浅绿到白色过渡背景。
不要纯白大空白。

5. 卡片：
白色卡片，圆角 20px 左右，轻阴影。

6. 输入框：
圆角 14px-16px，浅边框，聚焦时主色边框。
高度保持移动端舒适点击区域。

7. 图标：
使用线性图标或简洁面性图标。
不要使用过度卡通化、幼儿园风格的图标。

8. 页面整体要像“儿童健康管理平台入口”，不要像后台登录页。
