# 项目名称

孕产妇保健管理系统

# 页面名称

外院报告详情（归档展示页）

# 页面形式

Element Plus Dialog 弹框

不是独立页面

弹框宽度：

90vw

弹框高度：

85vh

支持内容滚动

---

# 页面定位

该页面用于查看已经导入归档的外院检验报告。

注意：

这是归档查看页面。

不是OCR识别页面。

不是OCR校验页面。

不允许展示：

* 自动匹配
* 匹配状态
* 匹配来源
* OCR统计信息
* OCR校验功能

用户只能查看。

---
