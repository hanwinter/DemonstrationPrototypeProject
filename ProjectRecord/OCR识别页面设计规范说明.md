
# 页面设计规范说明（产品设计文档）

# 外院检验报告OCR识别校验工作台设计规范

## 1. 页面定位

OCR识别校验工作台用于完成外院检验报告的识别、校验、匹配及导入工作。

系统通过OCR识别检验报告内容后，将识别结果展示在校验工作台中，由医务人员进行人工审核和修正，最终导入孕产妇保健管理系统。

设计目标：

* 提高录入效率
* 降低人工录入工作量
* 提高检验结果准确率
* 支持高频业务场景

---

# 2. 页面布局规范

页面采用：

**工作台（Workbench）布局模式**

布局结构：

```text
患者信息区

OCR图片区 + OCR校验工作台

操作区
```

整体内容区域居中显示。

---

# 3. 页面宽度规范

设计宽度：

```text
1800px
```

最小宽度：

```text
1600px
```

布局规则：

| 分辨率       | 显示效果      |
| --------- | --------- |
| 1920×1080 | 完整显示      |
| 2560×1440 | 居中显示，两侧留白 |
| ＜1600px   | 出现横向滚动条   |

样式规范：

```css
max-width:1800px;
min-width:1600px;
margin:0 auto;
```

---

# 4. 工作区布局规范

默认布局：

| 区域    | 比例  |
| ----- | --- |
| OCR原图 | 30% |
| 校验工作台 | 70% |

支持拖拽调整。

支持折叠OCR区域。

拖拽范围：

```text
30%~0%
```

折叠后：

```text
OCR区域隐藏
校验区域100%
```

支持：

* 图片缩放
* 图片滚动
* 图片全宽展示

缩放级别：

```text
50%
75%
100%
125%
150%
200%
```

默认：

```text

```

# 6. 校验工作台规范

## 6.1 统计信息

统计信息放置于工作台顶部。

展示内容：

* 识别项目数
* 自动匹配数
* 人工匹配数
* 未匹配数

采用标签化展示。

避免占用过多垂直空间。

---

## 6.2 报告信息区

展示：

* 报告类型
* 检查医院
* 检查时间
* 报告编号
* 项目组
* 备注

采用紧凑表单布局。

# 10. 新增行规则

新增按钮位于表格顶部。

新增规则：

## 未选中行

新增记录插入第一行。

## 已选中行

新增记录插入当前选中行上方。

禁止插入表格底部。

设计目的：

减少上下滚动。

提高录入效率。

---
