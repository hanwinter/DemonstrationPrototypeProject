# 外院报告影像与档案补录字段、状态说明

## 一、外院报告影像页面

页面路径：`/external-report-imaging`

该页面面向门诊医生，包含“外院报告采集”和“外院报告查阅”两个业务区域。

### 1. 外院报告采集字段说明

| 字段 | 说明 | 录入/展示方式 | 备注 |
| --- | --- | --- | --- |
| 孕产妇姓名 | 当前就诊孕产妇姓名 | 系统展示 | 来自当前档案上下文 |
| 年龄 | 当前孕产妇年龄 | 系统展示 | 用于医生核对身份 |
| 档案号 | 孕产妇保健档案编号 | 系统展示 | 外院报告归档的主关联编号 |
| 身份证号 | 孕产妇身份证号 | 系统展示 | 脱敏展示 |
| 孕周 | 当前孕周 | 系统展示 | 用于判断报告与产检阶段是否匹配 |
| 预产期 | 当前孕产妇预产期 | 系统展示 | 用于辅助核对 |
| 联系电话 | 当前孕产妇联系电话 | 系统展示 | 脱敏展示 |
| 报告类型 | 外院报告所属类型 | 单选按钮 | 可选：检验报告、NT报告、B超报告、心电图报告、唐筛报告、无创DNA报告、胎心监护报告、其他报告 |
| 报告日期 | 外院报告出具日期 | 日期选择 | 记录报告本身发生日期 |
| 检查医院 | 出具外院报告的医院 | 文本输入 | 用于后续查阅和归档核对 |
| 关联产检日期 | 该报告关联的本院产检日期 | 日期选择 | 用于把外院报告挂接到对应产检记录 |
| 备注 | 对该报告的补充说明 | 多行文本 | 可记录报告来源、特殊情况、缺页说明等 |
| 当前采集报告类型 | 当前高拍仪正在采集的报告类型 | 系统展示 | 随“报告类型”变化 |
| 报告图片 | 高拍仪采集到的报告图片页 | 图片预览 | 支持查看、旋转、删除、重拍、上移、下移 |

### 2. 外院报告查阅字段说明

#### 筛选条件

| 字段 | 说明 | 备注 |
| --- | --- | --- |
| 报告日期 | 按报告出具日期范围筛选 | 当前演示页面保留筛选入口 |
| 归档日期 | 按报告归档日期范围筛选 | 当前演示页面保留筛选入口 |
| 上传人 | 按上传人员筛选 | 支持文本输入 |
| 报告类型 | 按报告类型筛选 | 使用报告类型字典 |
| 报告状态 | 按报告当前状态筛选 | 可选：全部、正常、草稿、已作废 |

#### 报告列表

| 字段 | 说明 |
| --- | --- |
| 报告日期 | 外院报告出具日期 |
| 报告类型 | 外院报告所属类型 |
| 检查医院 | 出具报告的外院机构 |
| 页数 | 该报告影像页数 |
| 上传人 | 上传或采集该报告的人员 |
| 归档时间 | 报告正式归档时间 |
| 状态 | 报告在医生侧影像管理中的当前状态 |
| 操作 | 支持查看、补拍、作废 |

### 3. 外院报告影像状态说明

| 状态 | 定义 | 可执行操作 | 后续流向 |
| --- | --- | --- | --- |
| 草稿 | 报告图片或报告信息已临时保存，但尚未正式归档 | 可继续采集、保存归档 | 保存归档后转为正常 |
| 正常 | 报告已完成采集并正式归档，可供医生查阅和档案室补录 | 查看、补拍、作废 | 可进入档案补录流程 |
| 已作废 | 报告被确认无效，不再作为有效报告使用 | 查看历史影像 | 不参与有效业务处理 |

### 4. 主要按钮说明

| 按钮 | 说明 |
| --- | --- |
| 拍照 | 调用高拍仪采集当前报告页，生成新的报告图片 |
| 重拍 | 对当前采集页重新拍摄 |
| 保存草稿 | 保存当前报告信息和图片，但不正式归档 |
| 保存归档 | 将当前报告作为正式外院报告归档 |
| 返回档案 | 返回门诊档案上下文 |
| 新增外院报告 | 从查阅列表切换到采集页，新建外院报告 |
| 查看 | 打开报告影像查看窗口 |
| 补拍 | 切换到采集页，并带入该报告基础信息，用于补充拍摄缺失页 |
| 作废 | 将报告状态置为已作废 |

## 二、档案补录页面

页面路径：`/archive-supplement-workbench`

该页面面向档案室护士，用于对已经归档的外院报告进行结构化补录。页面以“产妇”为主层级，产妇下展示多份不同类型报告。

### 1. 筛选字段说明

| 字段 | 说明 | 备注 |
| --- | --- | --- |
| 归档日期 | 按报告归档日期范围筛选 | 取报告归档时间中的日期部分 |
| 处理状态 | 按补录处理状态筛选 | 可选：待录入、录入中、已录入、已作废 |
| 上传医生 | 按上传报告的医生筛选 | 来源于报告上传人字段 |
| 产妇档案号 | 按产妇档案号筛选 | 支持档案号片段匹配 |

### 5. 档案补录状态说明

档案补录页面不设置复核流程，因此不使用“已复核”状态。

| 状态 | 定义 | 产生方式 | 统计规则 |
| --- | --- | --- | --- |
| 待录入 | 报告已进入档案补录任务，但护士尚未开始录入结构化信息 | 报告归档后生成补录任务 | 计入待录入数量 |
| 录入中 | 护士已经保存过补录内容，但尚未确认完成 | 点击“保存”后，若原状态为待录入，则转为录入中 | 不计入已完成 |
| 已录入 | 护士确认当前报告补录完成 | 点击“完成录入” | 计入已完成、今日已处理 |
| 已作废 | 报告无效或不再参与补录 | 医生侧或业务侧作废 | 不计入有效报告总数和完成分母 |

### 6. 产妇任务状态规则

| 产妇任务状态 | 判定规则 |
| --- | --- |
| 已作废 | 该产妇名下报告全部为已作废 |
| 已录入 | 该产妇名下全部有效报告均为已录入 |
| 录入中 | 该产妇名下至少一份有效报告为录入中，且未全部已录入 |
| 待录入 | 不满足以上条件，且仍存在待录入有效报告 |

### 7. 档案补录按钮说明

| 按钮 | 说明 |
| --- | --- |
| 保存 | 保存当前补录内容；若当前报告为待录入，则状态变为录入中 |
| 完成录入 | 确认当前报告补录完成，状态变为已录入 |
| 新增一行 | 检验报告补录时新增一条检验项目 |
| 删除 | 检验报告补录时删除当前检验项目 |
