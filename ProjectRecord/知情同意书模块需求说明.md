# 知情同意书模块需求说明

## 1. 功能范围

- 知情同意书列表及筛选
- 新建和编辑知情同意书
- 病情摘要、风险告知模板选择和维护
- 患者扫码签字模拟
- 患者本人签字和家属代签
- 医生签字
- 文书归档
- 文书打印预览
- 文书作废

## 2. 状态模型

文书生命周期和签字进度分开处理，不能把“待患者签字”“待医生签字”作为文书生命周期状态。

### 2.1 文书生命周期状态

| 页面状态 | 含义 |
| --- | --- |
| 未归档 | 文书尚未归档，可根据签字情况执行编辑、签字和归档操作 |
| 已归档 | 文书已完成归档，正文和签字信息锁定，只允许查看、打印或作废 |
| 已作废 | 文书失效，保留原始正文、签字信息和作废原因，只允许查看 |

### 2.2 签字进度

- 患者/家属签字状态分为“未签字”和“已签字”。
- 医生签字状态分为“未签字”和“已签字”。
- 家属代签时需要记录代签人姓名及与孕妇关系。
- 文书处于“未归档”状态，且患者/家属和医生均已签字时，才允许归档。

患者签字和医生签字没有强制先后顺序，但归档前必须全部完成。

### 2.3 操作规则

列表操作顺序固定为：编辑、查看、医生签字、患者签字、归档、打印、作废。操作不因状态隐藏，不可执行时显示禁用态。

| 操作 | 可用条件 | 执行结果 |
| --- | --- | --- |
| 编辑 | 未归档，且患者和医生均未签字 | 修改文书正文和基本信息 |
| 查看 | 所有状态 | 只读预览文书和当前状态 |
| 医生签字 | 未归档，且医生尚未签字 | 写入医生签名和签字时间 |
| 患者签字 | 未归档，且患者/家属尚未签字 | 进入二维码及移动端模拟签字流程 |
| 归档 | 未归档，且双方均已签字 | 状态改为已归档，记录归档时间并锁定内容 |
| 打印 | 仅已归档 | 打开文书打印预览，打印后累计打印次数 |
| 作废 | 未归档或已归档 | 必填作废原因，状态改为已作废并保留原始数据 |

补充规则：

- 已有任一方签字后禁止编辑，避免签字内容和实际文书不一致。
- 已归档文书禁止编辑和继续签字。
- 已作废文书禁止编辑、签字、归档和打印。
- 预览页面只读，不提供返回编辑入口。
- 不提供批量打印，只允许对单份已归档文书打印。

## 3. 使用流程

### 3.1 创建文书

1. 进入知情同意书管理页。
2. 点击“新建知情同意书”。
3. 填写告知时间、告知医生、告知场景、病情摘要和风险告知。
4. 病情摘要和风险告知可以从模板中选择；模板支持新增、编辑和删除。
5. 设置允许的签字方式和知情确认文本。
6. 点击“预览”时只生成临时预览，不写入列表。
7. 点击“保存”后写入列表，生命周期为“未归档”。

### 3.2 患者或家属签字

1. 在未归档文书上点击“患者签字”。
2. 展示二维码、文书名称、患者姓名和有效时间。
3. 患者通过手机扫描二维码进入知情同意书查阅、签字页面。
4. 签字人阅读病情摘要、风险告知和知情确认文本。
5. 勾选知情确认。
6. 选择“孕妇本人”或“家属代签”。
7. 填写签字人姓名；选择家属代签时必须填写与孕妇关系。
8. 完成手写签名并提交。
9. 系统记录签字人、身份、关系、签名和签字时间。
10. 文书仍为“未归档”，患者签字进度显示已完成。

### 3.3 医生签字

1. 在未归档且医生尚未签字的文书上点击“医生签字”。
2. 核对医生姓名和签字时间。
3. 完成签名并确认。（使用ca进行签名）
4. 系统记录医生签名和签字时间。
5. 文书仍为“未归档”；双方均签字后，“归档”按钮变为可用。

### 3.4 归档

1. 点击“归档”。
2. 系统校验患者/家属签字和医生签字是否完整。
3. 任一签字缺失时不允许归档，并提示先完成双方签字。
4. 签字完整时显示归档确认。
5. 确认后状态改为“已归档”，记录归档时间。
6. 归档后正文和签字信息锁定，不允许修改。

### 3.5 打印

1. 在已归档文书上点击“打印”。
2. 打开文书打印预览。
3. 打印内容只包含文书正文和签字信息，不包含系统菜单及操作按钮。
4. 执行打印后累计该文书的打印次数。
### 3.6 作废

1. 对未归档或已归档文书点击“作废”。
2. 输入作废原因，原因不能为空。
3. 确认后状态改为“已作废”。
4. 保留文书正文、患者签字、医生签字、归档记录和作废原因。
5. 作废后只允许查看。

## 4. 列表和筛选要求

- 状态筛选支持“未归档、已归档、已作废”多选；未选择时显示全部。
- 支持按创建时间范围和创建医生筛选。
- 列表展示文书名称、告知场景、创建时间、创建医生、签字人、生命周期状态、签字进度、归档时间和操作。
- 签字进度分别显示患者/家属和医生是否已签字，不能用生命周期状态代替。
- 刷新操作在前端演示版中恢复初始 Mock 数据。

## 5. 文书展示和打印要求

- 预览和打印共用同一套正文内容，避免两个页面显示不一致。
- 医生签字和孕妇/家属签字在页面底部横向排列。
- 孕妇本人签字时不重复显示签字人字段。
- 家属代签时显示代签人姓名和与孕妇关系。
- 病情摘要和风险告知支持长文本自动换行和高度自适应。
- 已作废文书可查看原始内容，但不允许打印。
